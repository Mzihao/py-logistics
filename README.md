# 基于fastapi的物流聚合api服务

## version
python 3.9

## 安装库
pip install -r requirement.txt

## 本地运行
```shell
python app_run.py
```

## docker run
```shell
docker build -t fastapi_template:latest .

docker run -it -p 9000:9000 py-logistics
```

## 本地预览
http://127.0.0.1:9000/docs
